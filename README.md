# Google Cloud Skills Boost Automation

This repository contains automation scripts for Google Cloud Skills Boost labs.

## Networking 101 Lab

This script automates:

- Custom VPC creation
- Subnet creation
- Firewall rule creation

## Usage

Open Cloud Shell and run:

```bash
curl -LO https://raw.githubusercontent.com/YOUR_USERNAME/GoogleCloudSkillsboost/main/Networking101.sh

chmod +x Networking101.sh

./Networking101.sh
